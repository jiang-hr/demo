# 2. Continuity

## Definition 0

A function $$f$$ is continuous at the point $$a$$, if for any neighborhood $$V (f (a))$$ of its value $$f (a)$$ at a there is a neighborhood $$U(a)$$ of a whose image under the mapping$$ f$$ is contained in $$V (f (a))$$.
